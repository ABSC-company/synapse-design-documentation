#### «Настройки склада»

- **Назначение**: административные параметры модуля, доступны роли «администратор склада».
- **Поля**:
  - порог «на исходе срока» — число дней до истечения, после которого партия получает статус «на исходе срока» и публикуется `WMS.СрокГодностиПриближается` (`WMS-FR-09`, `WMS-FR-17`);
  - режим проверки условий хранения при размещении партии — «блокирующий» / «предупреждающий» (используется в «Приёмке» и «Перемещениях», `WMS-FR-07`);
  - разрешение ручного выбора партии при выдаче — переключатель «строго FEFO» / «допускать ручной выбор» (влияет на форму «Списание» и выдачу в других модулях через `СписатьВРасход`);
  - настройки принтера этикеток по умолчанию — выбор модели/драйвера принтера, шаблон этикетки (используется в «Приёмке», карточках партии/ТМЦ, `WMS-FR-20`);
  - настройки инвентаризации — периодичность и срок предварительного напоминания (используется на экране «Инвентаризация», см. 7.13).
- **Валидация**: порог «на исходе срока» — целое число ≥ 0.

**Wireframe:**
```
┌──────────────────────────────────────────────────────────┐
│  Настройки склада                                          │
├──────────────────────────────────────────────────────────┤
│  Порог «на исходе срока»          [30___] дней              │
│  Проверка условий хранения        ○ Блокирующий ○ Предупр.  │
│  Выбор партии при выдаче          ○ Строго FEFO ○ Ручной    │
│  Принтер этикеток по умолчанию    [Zebra ZD421 ▾] [Шаблон▾] │
│  Периодичность инвентаризации     [Ежеквартально ▾]         │
│  Напоминание за                   [7___] дней до начала     │
├──────────────────────────────────────────────────────────┤
│                                          [Сохранить]        │
└──────────────────────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Радио-группы («Блокирующий/Предупреждающий», «Строго FEFO/Ручной») — с коротким пояснением под каждым вариантом (одна строка), так как выбор напрямую влияет на поведение других экранов (Приёмка, Списание) и должен быть понятен без обращения к документации.
- Раздел виден и редактируется только ролью «администратор склада» (см. §7.0 про видимость по ролям).
