### Сайдбар склада

- **Левое навигационное меню** (сверху вниз): «Обзор склада», «Товары», «Поставщики», «Места хранения», «Партии и остатки», «Приёмка», «Перемещения», «Списание», «ТМЦ», «Аналитика склада», «Инвентаризация», «Настройки склада».
  - Пункт «Настройки склада» видим только роли «администратор склада» (см. `WMS-FR-17`).
  - Активный пункт подсвечен; при наличии непрочитанных уведомлений по разделу (например, просроченная инвентаризация) — точка-индикатор на пункте «Инвентаризация».
- **Шапка**: имя пользователя, роль (кладовщик / администратор склада / МОЛ / складской руководитель), селектор текущего склада/подразделения (если пользователь имеет доступ к нескольким складам — переключение обновляет весь контекст экранов), кнопка выхода.
- **Быстрые карточки вверху** (KPI-виджеты, дублируют часть главного экрана в свёрнутом виде):
  - «Остаток товаров» — суммарное количество позиций/партий в остатке;
  - «Партии на исходе срока» — счётчик партий в статусе «на исходе срока» (клик → «Партии и остатки» с активным фильтром);
  - «Текущие процессы» — сумма незавершённых документов приёмки + перемещений (клик → соответствующий раздел с фильтром «в работе»);
  - «Ближайшая инвентаризация» — дата и место ближайшей запланированной/просроченной сессии (клик → карточка сессии).
- **Панель уведомлений** (колокольчик в шапке): новое сообщение/событие, напоминание о необходимости инвентаризации (плановое и просроченное), предупреждения по условиям хранения (место вышло за температурный/влажностный порог), уведомление о партии, готовой к списанию по сроку годности (требует подтверждения ответственного, см. «Принятые решения»). Каждое уведомление кликабельно и ведёт к соответствующему объекту/экрану.

**Примечания для дизайнера:**
- Пункт «Инвентаризация» получает точку-индикатор (●) при наличии просроченной/предстоящей сессии — не бейдж-число, а именно точка (тип уведомления важнее количества).
- Быстрые карточки — одна строка из 4 плиток, на узких экранах сворачиваются в горизонтальный скролл, не в 2×2 (частота обращения высокая, важно не терять их из первого экрана).
- Селектор склада в шапке — если у пользователя доступ к одному складу, элемент скрывается (не показываем select с одним вариантом).
- Колокольчик показывает счётчик непрочитанных; открытие панели — dropdown, не отдельная страница.
