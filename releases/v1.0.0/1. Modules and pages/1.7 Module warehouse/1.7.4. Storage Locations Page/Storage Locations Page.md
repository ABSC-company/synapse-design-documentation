### Справочник «Места хранения»

- **Назначение**: модель «место хранения / подразделение» с иерархией (см. «Принятые решения»).
- **Вид**: дерево с возможностью переключения в плоский список; колонки — *Наименование, Тип (склад/секция/стеллаж/ячейка и т.п.), Родитель, Условия (темп/влажность), Ответственное подразделение, МОЛ*.
- **Карточка места хранения**:
  - наименование, тип, родительский узел (select с деревом, позволяет выбрать любой узел выше по иерархии);
  - условия хранения: температура мин/макс, влажность мин/макс (используются при проверке соответствия партии месту — `WMS-FR-07`);
  - ответственное подразделение;
  - материально ответственное лицо (МОЛ) — сотрудник (`HR`).
- **Действия**: «Создать» (с обязательным выбором родителя, кроме корневых узлов), «Редактировать», «Назначить МОЛ» (модальное окно выбора сотрудника).
- **Валидация**: наименование обязательно в рамках родителя; темп_мин ≤ темп_макс; циклические ссылки родитель→потомок запрещены.
- **Edge cases**: удаление/архивация места с ненулевым остатком партий — блокируется с сообщением «В месте есть остатки, перенесите их перед архивацией».

**Wireframe:**
```
┌────────────────────────────────────────────────────────────┐
│  Места хранения                    [Дерево|Список] [+ Создать]│
├────────────────────────────────────────────────────────────┤
│ ▾ 📍 Склад №1                         Темп 2–8°C   МОЛ: —   │
│    ▾ 📍 Секция «Холодовая цепь»       Темп 2–8°C   МОЛ: Ив. │
│         📍 Стеллаж А1                 Темп 2–8°C   МОЛ: Ив. │
│    ▸ 📍 Секция «Расходники»           Темп 15–25°C МОЛ: Пет.│
│ ▸ 📍 Аптека №2                                               │
└────────────────────────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Дерево — раскрывающиеся узлы (▾/▸), отступ по уровню иерархии; условия хранения показываются справа компактно, чтобы были видны без раскрытия карточки.
- «Создать» с предвыбранным родителем — если пользователь нажал «+» рядом с узлом дерева, модалка создания открывается с уже подставленным родителем.
- Место без назначенного МОЛ — прочерк «—», не пустая ячейка (чтобы не путать с ошибкой загрузки).
