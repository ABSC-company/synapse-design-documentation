#### «Сканирование» (кросс-модульный режим)

- **Назначение**: реализует публичный метод `НайтиПоШтрихкоду` (см. §9) как основу обоих режимов.
- **Режим 1 — «Быстрый поиск»**: поле ввода/фокус для сканера → после скана система ищет объект (партию или ТМЦ) по штрихкоду/QR и показывает карточку найденного объекта (наименование, учётное место, закреплён за) без создания какого-либо документа. Если объект не найден — сообщение «Объект не найден» без побочных эффектов.
- **Режим 2 — «Инвентаризация по месту»**: доступен только внутри активной сессии инвентаризации (см. 7.13); выбор места хранения/помещения → «Начать сессию» → список сканирований в реальном времени (объект, ожидаемое место, результат: совпадает/расхождение/—) → «Завершить сессию» формирует отчёт по расхождениям.
- **Действие «Печать этикетки»** доступно из карточки найденного объекта в обоих режимах (`WMS-FR-20`).
- **Технические ограничения**: текущая реализация ориентирована на стационарный сканер и стандартный принтер этикеток; предпочтительный формат штрихкода — QR-код (см. «Принятые решения»).

**Wireframe:**
```
Режим 1 — Быстрый поиск:
┌──────────────────────────────────────────┐
│  🔦 Отсканируйте объект: [_____________]   │
├──────────────────────────────────────────┤
│  ✅ Найдено:                               │
│  A-1042 · Бинт стерильный                 │
│  Учётное место: Склад №1 / Стеллаж А1     │
│  Закреплён за: —                          │
│                        [🖨 Печать этикетки]│
└──────────────────────────────────────────┘

Не найдено:
┌──────────────────────────────────────────┐
│  ⚠ Объект не найден                       │
│  Штрихкод не соответствует ни одной        │
│  партии/ТМЦ в системе                     │
└──────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Режим 1 доступен из любого места интерфейса (глобальная горячая клавиша/иконка сканера в шапке), не только со страницы «Сканирование» — часто используется «на ходу» кладовщиком.
- Результат поиска — компактная карточка без действий по редактированию (это read-only режим, кроме кнопки печати этикетки); долгое нажатие/клик на найденный объект может вести в полную карточку партии/ТМЦ.
- «Не найдено» — нейтральный (не ошибка сервера) стейт: жёлтая/серая иконка предупреждения, не красный крест.
