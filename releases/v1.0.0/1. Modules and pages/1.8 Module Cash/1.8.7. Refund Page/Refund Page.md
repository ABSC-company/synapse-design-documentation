#### «Возврат»

- **Назначение**: `Refund` по сумме или по конкретной услуге (`CASH-FR-11`, `CASH-FR-24`).
- **Состав**: выбор оплаченного платежа/чека → переключатель «По сумме» / «По услугам»; поле причины; кнопка «Провести возврат» → печать чека-возврата.

**Wireframe:**
```
┌────────────────────────────────────────────────────┐
│  Возврат · Визит №10231 · Юсупова Н.Б.           [✕]│
│  [ По сумме ]   [ По услугам ]                        │
├────────────────────────────────────────────────────┤
│  Вариант «По услугам»:                                │
│  ☑ Анализ крови ОАК         170 000                   │
│  ☐ Консультация терапевта   150 000                   │
│                          Сумма возврата: 170 000      │
│  Причина * [_____________________________]           │
├────────────────────────────────────────────────────┤
│                          [Отмена]  [Провести возврат] │
└────────────────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Вкладка «По сумме» — простое числовое поле с ограничением «не более оплаченного остатка», показать инлайн-ошибку при превышении, а не только при попытке сабмита.
- Вкладка «По услугам» — чекбоксы на позициях счёта, сумма возврата считается и обновляется вживую при отметке/снятии галочки.
- После проведения — печатается чек-возврат (визуально отличается от обычного чека — например, красная плашка «ВОЗВРАТ» в шапке), и в списке 6.2 визит может снова появиться со статусом «частично» на новую сумму долга.
