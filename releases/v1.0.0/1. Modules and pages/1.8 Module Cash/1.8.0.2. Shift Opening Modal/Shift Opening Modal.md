####  «Открытие смены»

- **Назначение**: обязательный первый шаг (`CASH-FR-01`, `CASH-FR-02`).
- **Поля**: сумма размена наличных (число), рабочее место (авто по логину/выбор из списка), кассир (авто из сессии `AUTH`).
- **Действие**: «Открыть смену» — создаёт `Shift`, `ZReport` и первый `XReport` дня.

**Wireframe:**
```
┌──────────────────────────────────────────┐
│          🔓 Открытие смены                │
├──────────────────────────────────────────┤
│  Кассир          Иванов И.И.  (из сессии) │
│  Рабочее место   [Касса №1 ▾]             │
│  Сумма размена * [_______] UZS            │
├──────────────────────────────────────────┤
│                       [Открыть смену]     │
└──────────────────────────────────────────┘
      ▲ модальное окно поверх всего интерфейса кассы
```

**Примечания для дизайнера:**
- Окно нельзя закрыть крестиком/по Esc/кликом вне — единственный выход — заполнить и нажать «Открыть смену».
- Если рабочее место однозначно определяется по логину/устройству — селект скрывается, показывается как текст.
- После открытия — прямой переход на «Список ЭМК к оплате» (6.2), а не на промежуточный пустой экран.
