#### «Перерыв / Пересменка» (закрытие X-отчёта)

- **Назначение**: передача рабочего места без прерывания смены (`CASH-FR-12`, `CASH-FR-20`).
- **Состав**: кнопка «Закрыть X-отчёт (перерыв / пересменка)» постоянно доступна на рабочем экране кассира; предпросмотр текущих итогов X-периода → печать → возврат к состоянию «касса свободна».

**Wireframe:**
```
┌──────────────────────────────────────────┐
│   Закрыть X-отчёт?                        │
│   Период: 09:00–14:20                      │
├──────────────────────────────────────────┤
│  Способ       Сумма      Чеков            │
│  Наличные     840 000     6               │
│  Терминал     260 000     2               │
│  Payme        150 000     1               │
├──────────────────────────────────────────┤
│           [Отмена]      [Закрыть и печать]│
└──────────────────────────────────────────┘

После печати:
┌──────────────────────────────┐
│  ✅ X-отчёт напечатан          │
│  Касса свободна для входа     │
│  следующего кассира           │
└──────────────────────────────┘
```

**Примечания для дизайнера:**
- Кнопка располагается в постоянно видимой зоне (шапка/боковая панель рабочего экрана кассира), а не спрятана в меню — используется многократно за смену.
- После печати — экран логаута/готовности к новому кассиру, но смена/Z-отчёт продолжаются в фоне без визуальных признаков «остановки» учёта.
- Итоги показываются только за текущий X-период (не нарастающим итогом за день) — подписать это явно («Период: 09:00–14:20»), иначе легко перепутать с Z-отчётом
