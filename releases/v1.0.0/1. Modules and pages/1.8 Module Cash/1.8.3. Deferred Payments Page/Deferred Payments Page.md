#### «Отложенные платежи»

- **Назначение**: возврат к платежам, отложенным в рамках текущей смены (`CASH-FR-09`).
- **Состав**: список (*Пациент, № визита, Сумма, Кассир, Время откладывания*); действие «Вернуться к оплате».

**Wireframe:**
```
┌──────────────────────────────────────────────────────────┐
│  Отложенные платежи                                        │
├──────────────────┬────────┬─────────┬─────────┬────────────┤
│ Пациент          │№визита │ Сумма   │ Кассир  │ Отложено в │
├──────────────────┼────────┼─────────┼─────────┼────────────┤
│ Каримов А.А.     │ 10245  │ 450 000 │ Иванов  │ 13:40      │[Вернуться к оплате]│
└──────────────────┴────────┴─────────┴─────────┴────────────┘
```

**Примечания для дизайнера:**
- Список scope-ится текущей сменой (не всей историей) — при закрытии смены отложенные платежи не переносятся автоматически в список следующей смены как «отложенные», они просто остаются видимыми в общем «ЭМК к оплате» по задолженности.
- Пустое состояние — «Отложенных платежей нет» с иконкой раздела.
