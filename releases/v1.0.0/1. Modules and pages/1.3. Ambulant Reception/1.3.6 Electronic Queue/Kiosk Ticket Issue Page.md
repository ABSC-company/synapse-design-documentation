## Kiosk — терминал выдачи талонов

Экран расположен в зоне ожидания, рассчитан на самостоятельное использование пациентом без помощи персонала.

**Состояние по умолчанию**
Один крупный элемент по центру экрана — кнопка «Получить очередь в регистратуру». Больше на экране ничего не должно отвлекать: пациент должен понять, что делать, за 1–2 секунды.

**После нажатия**
Короткая пауза с индикатором, пока печатается чек.

**Талон выдан**
На экране крупно показывается номер, который только что напечатан («Ваш номер: A-014»), чтобы пациент мог свериться с чеком. Через несколько секунд экран сам возвращается в состояние по умолчанию.

**Чек не печатается**
Сообщение об ошибке и предложение обратиться к регистратору напрямую — киоск не должен становиться тупиковым сценарием для пациента.

На чеке: номер талона, дата и время выдачи.
