Компонент «Электронная очередь» — фиксированный правый блок на страницах модуля Регистратуры.

Он может занимать всю доступную высоту контента или сочетаться с соседним компонентом в одной обёртке, например с календарём.

Основная структура компонента:

- Header
  - Название очереди, например: «Регистратура. Окно 3».
  - Список окон регистратуры, из которого пользователь выбирает текущее рабочее окно.
  - Две основные кнопки‑триггеры:
    - приостановка очереди;
    - вызов следующего посетителя.

- Body
  - Список карточек текущей электронной очереди.
  - Каждая карточка содержит:
    - порядковый номер;
    - время становления в очередь;
    - время ожидания.

Требования к оформлению:

- Блок должен быть визуально отделён от основного контента страницы.
- Заголовок очереди должен быть чётким и легко читаемым.
- Кнопки управления очередью должны быть доступны и заметны.
- Список карточек должен быть компактным, но информативным, с акцентом на номер и текущее ожидание.

Варианты использования:

- Полноэкранный вид: когда «Электронная очередь» занимает всю высоту доступной контентной области.
- Комбинированный вид: когда компонент расположен рядом с дополнительным модулем, например календарём, в единой обёртке.

Этот виджет — лишь часть более широкой системы электронной очереди, включающей киоск выдачи талонов, публичное табло вызова и табло у кабинетов врачей. Полное описание сквозного сценария см. в [1.3.6. Electronic Queue](../../1.3.6%20Electronic%20Queue/Electronic%20Queue.md).
