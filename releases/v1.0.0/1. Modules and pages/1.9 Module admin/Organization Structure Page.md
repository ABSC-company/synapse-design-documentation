## Организация / Оргструктура

**Реквизиты организации**
Наименование, реквизиты, часовой пояс, язык по умолчанию — заполняются в мастере настройки, редактируются здесь.

**Дерево структурных единиц**
Филиал → корпус → этаж → отделение, произвольная глубина вложенности. Добавление и редактирование узлов дерева прямо на этой странице.

---

### Заметки для макетов

- Это дерево не локальная деталь одной страницы — на него ссылаются другие модули (например, «этаж» в компоненте «Электронная очередь», мультикассовый режим в «Кассе»). Каждый узел должен явно нести свой тип (филиал/корпус/этаж/отделение), чтобы в других модулях при выборе из этого дерева не приходилось гадать, что перед пользователем.
- Дерево может быть глубоким и разветвлённым — предусмотреть сворачивание веток и, возможно, поиск по названию узла, а не только визуальный скролл.
- Удаление узла, на который уже ссылаются другие модули (например, кабинет с привязанным табло очереди), — потенциально разрушительное действие: нужен явный предупреждающий диалог, а не тихое удаление.
