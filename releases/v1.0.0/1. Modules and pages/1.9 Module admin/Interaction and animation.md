## Взаимодействие и анимация

- Мастер первичной настройки — линейный прогресс, шаги нельзя перепрыгнуть, кроме явно помеченного опциональным шага 5.
- Отзыв сессии не разрывает соединение мгновенно: пользователь видит предупреждение с обратным отсчётом в реальном времени и может успеть среагировать до фактического завершения.
- Уведомление о подключении техподдержки приходит пользователю сразу в момент подключения, а не постфактум.
- Изменения в матрице прав роли применяются без полной перезагрузки страницы; состояние чекбоксов сохраняется оптимистично.
- Экран с резервными кодами 2FA нельзя закрыть случайно (клик мимо, Esc) — только явным подтверждением «Я сохранил коды».
- Диалоги подтверждения для необратимых действий (архивация роли, откат статуса, сброс пароля, отзыв сессии) — модальные, требуют осознанного действия, а не просто повторного клика.
