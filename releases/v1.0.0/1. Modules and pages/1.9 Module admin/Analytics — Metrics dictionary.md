## 5. KPI-блоки

### Первый ряд — операционные показатели

| KPI | Что показывает |
|---|---|
| Пациенты за период | Общее количество уникальных пациентов, обратившихся за период |
| Новые пациенты | Пациенты, впервые обратившиеся в клинику за период |
| Повторные пациенты | Пациенты, уже имевшие визиты до начала периода |
| Визиты | Общее количество визитов за период |
| Завершённые визиты | Визиты со статусом «Завершено» |
| Отменённые визиты | Визиты со статусом «Отменено» |
| Записи на приём | Количество созданных записей (включая будущие) за период |
| No-show / неявки | Количество визитов, на которые пациент не явился |
| Среднее время ожидания | Среднее время от постановки в электронную очередь / записи до начала приёма |
| Пациенты в электронной очереди | Текущее количество ожидающих в очереди (значение в реальном времени, не за период) |

### Второй ряд — финансовые показатели

| KPI | Что показывает |
|---|---|
| Выручка за период | Общая начисленная выручка за период |
| Оплачено | Сумма фактически поступивших оплат за период |
| Остаток задолженности | Сумма неоплаченных начислений на конец периода |
| Возвраты | Сумма возвратов за период |
| Средний чек | Выручка за период / количество завершённых визитов |
| Загрузка врачей | Средний коэффициент загрузки врачей клиники за период, % |
| Загрузка отделений | Средний коэффициент загрузки отделений за период, % |
| Топ услуг по количеству | Ссылка-переход к таблице топ услуг по количеству |
| Топ услуг по выручке | Ссылка-переход к таблице топ услуг по выручке |

Каждая KPI-карточка содержит:

- Название показателя.
- Крупное числовое значение.
- Сравнение с предыдущим аналогичным периодом: стрелка вверх/вниз и % изменения.
- Цвет индикатора изменения зависит от смысла показателя (для «Отменённые визиты» и «No-show» рост — это плохо и подсвечивается предупреждающим цветом, для «Выручка» и «Завершённые визиты» рост — это хорошо и подсвечивается положительным цветом).
- Клик по карточке открывает детализацию (см. Flow 9).

Показатели «Пациенты в электронной очереди» и часть операционных данных обновляются ближе к реальному времени и визуально помечены отдельной меткой (например, точка-индикатор «live»), чтобы не путать их с показателями за выбранный период.
