#### «Направление к врачу»

- **Назначение**: модальное окно оформления направления пациента к другому врачу из открытой консультации (`PHYS-FR-03`, `PHYS-FR-04`).
- **Состав**:
  - Выбор целевого врача (с индикацией занятости, как в «Booking Page») — обязателен, выбора «только специализация» нет.
  - Поле «Причина направления» — обязательно.
  - Информационный блок: пояснение, что визит в клинику и номерок электронной очереди сохраняются, закрывается только текущий приём.
- **Действия**:
  - «Отправить направление» — создаёт `Referral(создано)`, переводит его в `в_очереди`: текущий `Encounter` закрывается (`завершён`), у целевого врача создаётся новый `Encounter(ожидает)` с тем же `visit_id`/`talon_id`, талон переадресуется на кабинет/врача (см. «Электронная очередь»).
  - «Отмена» — закрывает окно без создания направления, консультация продолжается.

**Wireframe:**
```
┌───────────────────────────────────────────────┐
│ Направление к врачу                        [×]│
├───────────────────────────────────────────────┤
│ Целевой врач *                                 │
│ [🔍 Поиск врача________________________]       │

│  ○ Ахмедова Д. — Терапевт     🟡 занят до 10:40│
│  ○ Носирова М. — Невролог     🔴 нет мест сегодня│
│                                                 │
│ Причина направления *                          │
│ [_____________________________________]        │
│                                                 │
│ ℹ Визит в клинику и номерок очереди сохранятся,│
│   закроется только текущий приём               │

│                        [Отмена] [Отправить]    │
└───────────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Индикатор занятости целевого врача — переиспользовать визуальный паттерн из «Booking Page» (`1.3.2`), не изобретать новый.
- Врачи без свободных окон в этот день по-прежнему выбираемы (направление — не запись на конкретное время, а постановка в очередь к врачу), но индикатор должен честно показывать нагрузку, чтобы направляющий врач понимал ожидание пациента.
- После отправки окно закрывается, а текущая консультация показывает состояние «завершена направлением» — врач не остаётся в форме уже закрытого приёма.
- Поле «Причина направления» — обычный многострочный текст, без структурированного справочника (в отличие от целевого врача, который выбирается из списка).
