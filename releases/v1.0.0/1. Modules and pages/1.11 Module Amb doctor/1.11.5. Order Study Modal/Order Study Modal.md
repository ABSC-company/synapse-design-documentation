#### «Назначение исследования»

- **Назначение**: модальное окно заказа лабораторного и/или инструментального исследования прямо из консультации, без выхода в отдельный модуль (`PHYS-FR-05`, `PHYS-FR-06`).
- **Состав**:
  - Переключатель «Лабораторное / Инструментальное».
  - Список доступных исследований (из справочника `LAB`/`INSTR`), мультивыбор.
  - Итоговый список выбранных позиций перед отправкой заказа.
- **Действия**:
  - «Назначить» — вызывает `LAB.СоздатьЗаказ` и/или `INSTR.СоздатьЗаказ` (в зависимости от типа и состава выбора), в консультации сохраняется ссылка на заказ со статусом «назначено».
  - «Отмена» — закрывает окно без создания заказа.

**Wireframe:**
```
┌───────────────────────────────────────────────┐
│ Назначение исследования                    [×]│
├───────────────────────────────────────────────┤
│ ( ●Лабораторное )  ( ○Инструментальное )       │
│                                                 │
│ 🔍 Поиск по справочнику                        │
│ ☑ Общий анализ крови                           │
│ ☑ Биохимический анализ крови                   │
│ ☐ Общий анализ мочи                            │
│ ☐ Коагулограмма                                │
│                                                 │
│ Выбрано: 2 позиции                             │
│                                                 │
│                        [Отмена] [Назначить]    │
└───────────────────────────────────────────────┘
```

**Примечания для дизайнера:**
- Переключатель типа не сбрасывает уже выбранные позиции другого типа — врач может назначить и лабораторное, и инструментальное исследование за один проход, окно не закрывается между переключениями (два отдельных вызова `СоздатьЗаказ`, один UI-шаг).
- После назначения в форме консультации сразу появляется строка заказа со статусом «назначено» — окно не должно быть единственным местом, где виден факт назначения.
- Статус «результат готов» появляется автоматически по событию из `LAB`/`INSTR` — на этом экране нет и не должно быть действия «обновить статус вручную» или «импортировать результат».
- Справочник исследований — общий для лабораторных и инструментальных, но выводится раздельно по переключателю; поиск работает в пределах текущего выбранного типа.
