### Flow 11. Доступ с ограниченными правами

1. Руководитель подразделения открывает дашборд.
2. Система автоматически ограничивает данные его подразделением (без возможности переключиться на чужое через фильтр филиала).
3. Финансовый отдел видит полный финансовый блок, но не видит персональной детализации по пациентам, если у роли нет отдельного права на это.
4. Если у роли нет доступа к разделу целиком — вместо дашборда показывается состояние «Нет доступа» (см. §9).
