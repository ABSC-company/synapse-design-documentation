
### Flow 10. Экспорт отчёта

1. Пользователь нажимает кнопку «Экспорт» в Page Header.
2. Открывается компактное меню: формат (PDF / Excel), состав (только KPI / KPI + графики / полный дашборд с таблицами).
3. Пользователь подтверждает — запускается формирование файла, кнопка переходит в состояние «Экспорт выполняется».
4. По готовности файл скачивается автоматически, пользователь получает уведомление об успехе.
