### Clinic BI Analytics Dashboard Page

## 3. Пользовательские сценарии

### Flow 1. Первый экран — общая картина

1. Пользователь открывает Аналитическую панель.
2. Система применяет период по умолчанию (например, «7 дней») и последний использованный набор фильтров (если был сохранён).
3. Система показывает первый ряд KPI-карточек, второй ряд финансовых KPI, графики и таблицы.
4. Руководитель за 10–15 секунд считывает: сколько пациентов, визитов, выручки, есть ли предупреждения — без необходимости кликать куда-либо.
