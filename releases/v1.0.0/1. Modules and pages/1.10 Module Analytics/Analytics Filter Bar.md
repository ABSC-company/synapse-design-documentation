### Flow 2. Смена периода

1. Пользователь выбирает период в верхней строке фильтров: сегодня / вчера / 7 дней / 30 дней / месяц / квартал / год / произвольный.
2. При выборе «Произвольный период» открывается календарь с выбором диапазона дат.
3. Система обновляет все KPI, графики, таблицы и блок предупреждений под новый период.
4. Рядом с каждым KPI отображается сравнение с предыдущим аналогичным периодом (стрелка и % изменения).


### Flow 3. Фильтрация по филиалу, врачу, специализации

1. Пользователь открывает фильтр «Филиал / корпус / отделение» и выбирает нужный узел оргструктуры.
2. Список врачей и специализаций в остальных фильтрах сужается до выбранного подразделения.
3. Пользователь дополнительно выбирает врача и/или специализацию.
4. Система обновляет дашборд, оставляя выбранные фильтры видимыми как активные чипы над контентом.
5. Пользователь может снять отдельный фильтр кликом на чип или сбросить все фильтры одной кнопкой.
