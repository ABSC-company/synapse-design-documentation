## 6. Графики и виджеты

**График динамики визитов**
Линейный или столбчатый график по дням/неделям/месяцам (в зависимости от длины периода): общее количество визитов, с наложением завершённых и отменённых. Легенда над графиком, переключение группировки — рядом с заголовком графика.

**График выручки**
Линейный график рядом с графиком визитов, той же временной шкалой, чтобы графики можно было сопоставлять взглядом по вертикали. При наведении — тултип с разбивкой: начислено / оплачено / возврат за точку.

**Диаграмма по отделениям**
Круговая или столбчатая диаграмма распределения визитов (или выручки — переключатель) по отделениям текущего фильтра. Каждый сегмент кликабелен и ведёт к отфильтрованному дашборду по этому отделению.

Правило по количеству виджетов на графики: не более двух ключевых графиков в одном ряду и одной диаграммы распределения — дашборд не должен превращаться в стену из графиков, каждый виджет должен отвечать на один понятный вопрос.
