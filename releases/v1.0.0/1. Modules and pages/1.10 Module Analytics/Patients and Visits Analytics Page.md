### Flow 4. Анализ динамики визитов

1. Пользователь смотрит на график динамики визитов за выбранный период.
2. При наведении на точку/столбец график показывает точные цифры за день (всего визитов, завершённые, отменённые, no-show).
3. Пользователь может переключить группировку графика: по дням / по неделям / по месяцам — в зависимости от длины периода.
